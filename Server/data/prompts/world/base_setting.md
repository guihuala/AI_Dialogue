【学校设定：江城传媒大学（JMU）】
校名：江城传媒大学（Jiangcheng Media University，简称JMU）
性质：民办/公办二本（可据游戏定位调整），艺术类提前批招生
定位：“南方传媒人才的黄埔军校”（自封）、“网红孵化基地”（他称）
校训：知行合一，视听无界（学生调侃：作业做不完，熬夜无极限）

【校园格局】
- 主校区：位于江边，占地不大但设计感强。主楼是玻璃幕墙的现代建筑，学生戏称“透明自习室”（因为从外面能看见谁在玩手机）
- 传媒大道：连接教学楼和宿舍区的主干道，两侧是社团招新、作业展映、设备租借的海报墙，号称JMU的“信息流入口”
- 影视实训中心：学校最“贵”的地方，拥有专业影棚、非编机房、录音棚、虚拟演播厅。学生凭证件预约，但永远抢不到
- 图书馆：六层建筑，二三层是考研考公党长期驻扎地，四五层是编导生拉片室（配有蓝光播放机和隔音耳机），顶层是“天台沙龙”——学生拍作业、聊剧本、谈恋爱的多功能区

【宿舍区】
宿舍类型：
- 校内公寓：4人间，上床下桌，独立卫浴，空调（需单独交租），1200元/年
- 校外租房：大三后部分学生搬出，学校周边单间1500-2500元/月

宿管阿姨设定：
- 姓名：张桂芳
- 特点：眼神毒辣，能从你提的袋子里闻出有没有带外卖；记忆力惊人，能记住全楼200多号人的脸和班级；刀子嘴豆腐心
- 管理规定：晚上11点锁门，11点半记晚归，12点上报学院。违禁电器检查每周一次

【语言风格提示】
描述学校时使用学生视角的调侃语气，如“自封”、“戏称”、“号称”、“永远抢不到”等，突出理想与现实的落差感。
