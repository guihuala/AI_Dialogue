## Project Introduction

This project is an **AI Character Simulation Game** driven by large language models (LLMs), created as a graduation project by Mokukeki.

Players (assuming the role of Lu Chen Anran) will deeply interact with multiple AI roommates—each with distinct personalities and hidden secrets—across the four-year university cycle, making decisions to navigate academics, interpersonal relationships, and social life.

In my vision, an LLM-driven character simulation game should offer players more possibilities—thus I designed a custom prompt feature along with the ability to package and publish these prompts. The goals of this feature are:
- Write your own worldviews, characters, and special events.
- Write "skills" to instruct the AI to complete tasks you propose.
- Publish your compiled mod packs to the server for other players to experience.

## Quick Start

### Step 1: Backend Environment Setup
1. Navigate to the `Server` directory and install dependencies:
   ```bash
   cd Server
   pip install -r requirements.txt
   ```
2. Create a local `Server/.env` from `Server/.env.example`, then fill in your API key and admin password:
   ```env
   DEEPSEEK_API_KEY=your_api_key_here
   ADMIN_PASSWORD=change_me_to_a_strong_password
   ADMIN_SESSION_TTL_SECONDS=43200
   ```
   `Server/.env` is ignored by `.gitignore` and should not be committed.
3. Start the backend service:
   ```bash
   python src/app.py
   ```
   *Default listening address: `http://127.0.0.1:8000`*

### Step 2: Frontend Environment Setup
1. Navigate to the `WebClient` directory and install dependencies:
   ```bash
   cd WebClient
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
   *Access the local port shown in the console via your browser by default.*

### Admin Panel
- The admin entry is `http://localhost:5173/admin`
- Admin login is now verified by the backend using `ADMIN_PASSWORD`
- The backend issues a short-lived admin token; tokens expire after a server restart in the current design

## Tech Stack
- **Frontend**: React 18, Vite, Tailwind CSS, Lucide React, Framer Motion
- **Backend**: FastAPI (Python), ChromaDB (Vector DB), Pydantic
- **AI Engine**: GPT-4 / DeepSeek-V3 (OpenAI Protocol Compatible)
